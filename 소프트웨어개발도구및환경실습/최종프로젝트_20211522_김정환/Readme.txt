실행 환경 : visual studio를 기반으로 한 OpenFramework
maze 데이터는 실습 때 사용한 데이터와 동일

실행 방법
File 탭의 Open 버튼을 눌러 .maz 확장자를 가진 파일을 열어 미로가 그려질 수 있도록 한다.

그 후 View 탭에서 Show DFS와 Show BFS를 통해서 각각의 경우에 따라서 경로를 확인할 수 있다.

Open 파일로 다른 .maz 파일도 열어서 확인할 수 있는데, 이를 여러 번 반복하여 여러 파일을 열어본 뒤, View 탭의 Before 버튼을 눌러서 이전의 미로로 돌릴 수 있다. 이전의 미로로 돌리다가 cmd 창에서 먼저 파일을 열어야 한다는 문구가 나오면 더이상 이전 미로가 존재하지 않는 경우이므로 Open 버튼으로 파일을 추가해야 한다.

File 탭의 Login 버튼은 cmd 창에서 사용자의 이름을 입력받아 사용자 로그인을 실시한다. 이 경우에 사용자 별로 미로를 여는 경우에 stack에 저장하여 Before 버튼으로 이전 미로를 여는 데이터를 따로 저장한다. 
File 탭의 Logout 버튼은 로그인 하지 않은 상태, 즉 guest 상태로 전환되며 guest 상태에서는 다른 사용자로 로그인 시 불러왔던 미로의 기록이 저장되지 않는다.

File 탭의 Exit 버튼으로 언제든 프로그램을 종료할 수 있다.
